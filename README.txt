# Battleground Defender

A modern, high-performance World of Warcraft Addon designed for tactical communication in Battlegrounds. Developed as a successor to the legacy *BG Defender (based on AB Defender)*.

## Features
- **Modular Snap-Layout**: Drag action buttons (INC, HELP, etc.) to any side of the numeric core. They snap into place and stay anchored.
- **Elastic UI**: The background frame automatically resizes to fit your custom layout perfectly.
- **Auto-Event Reporting**: Automatically announces base captures, flag pickups, and orb control in the group chat.
- **Smart Localization**: Supports German and English with an option to force English output for international groups.
- **Media Integration**: Full Support for `LibSharedMedia-3.0`. Choose any font or sound registered in your UI.

## Commands
- `/bd` or `/bd help` - Displays the command list.
- `/bd open` - Toggles the visibility of the main frame.
- `/bd reset` - Resets all settings to factory defaults and reloads the UI.

## Installation
1. Download the repository.
2. Move the `BattlegroundDefender` folder into your `_retail_/Interface/AddOns/` directory.
3. Enjoy tactical superiority!
